# sunjjoygupta

Official portfolio website of **Sunjjoy Gupta** — Govt. Authorised Lean Consultant, Author & Corporate Trainer.

🌐 Live: [https://sunjjoygupta.com](https://sunjjoygupta.com)

## About
35+ years driving operational excellence across Japanese & Korean MNCs and Indian MSMEs. Expert in Lean Manufacturing, Continual Improvement, ISO 9000 & IATF 16949. Author of *"The Bridge"*.

## Tech
Static website — plain HTML, CSS & JavaScript. No build step.

| File | Purpose |
|------|---------|
| `index.html` | All page sections (Hero, About, Programs, Book, Team, Testimonials, Contact) |
| `styles.css` | Design system, layout, responsive + animations |
| `script.js` | Sticky nav, mobile menu, scroll reveals, stat counters, lead popup |
| `robots.txt`, `sitemap.xml` | SEO |
| `llms.txt` | GEO / AI-search optimization |
| `favicon.svg`, `site.webmanifest` | Branding / PWA |

## Run locally
```bash
# any static server works, e.g.
npx serve .
# or
node server.cjs   # serves on http://localhost:4321
```

## Features
- Fully responsive (mobile → desktop)
- Lead-capture popup → sends enquiry directly to WhatsApp
- SEO + local GEO (Pune) + structured data (JSON-LD)
- Clean SVG icons, no external icon libraries
